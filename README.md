## Installation

PSYLib is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'PSYLib', :git => 'https://github.com/ios-sungyoungpark/PSYLib.git', :tag => '0.1.0'
```
